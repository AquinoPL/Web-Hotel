# Sistema Web Hotel

## Descripción
Desarrollo de una página web para la gestión y reserva de habitaciones de un hotel. El sistema permite a los usuarios consultar la disponibilidad de habitaciones, visualización de los restaurantes y menu disponibles, realizar reservas en línea y almacenar la información de clientes y reservas en una base de datos MySQL. El proyecto fue desarrollado con enfoque en la funcionalidad, organización de datos y facilidad de uso.

## Tecnologías usadas
- Lenguajes: PHP, HTML, CSS
- Base de datos: MySQL
- Herramientas: XAMPP, Visual Studio Code
- Control de versiones: Git, GitHub

## Funcionalidades
- Registro de clientes
- Visualización de habitaciones disponibles
- Visualización de Carta de los restaurantes del Hotel
- Reserva de habitaciones en línea
- Almacenamiento de reservas y consumos en base de datos MySQL
- Validación básica de formularios
- Gestión de fechas de ingreso y salida

## Cómo ejecutar el proyecto
1. Instalar XAMPP o cualquier servidor local compatible con PHP y MySQL.
2. Clonar el repositorio desde GitHub.
3. Importar la base de datos MySQL usando phpMyAdmin.
4. Configurar las credenciales de conexión a la base de datos en el archivo PHP correspondiente.
5. Iniciar el servidor Apache y MySQL.
6. Acceder al proyecto desde el navegador web.
